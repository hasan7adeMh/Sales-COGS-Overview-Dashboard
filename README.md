**Overview**

This project is an interactive Sales & COGS Overview Dashboard built in Power BI to analyze sales performance, cost behavior, and profitability across products, segments, countries, and time. The dashboard combines financial KPIs with intuitive visuals and advanced interactivity to support data-driven business decisions.

**Tools & Technologies**

- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Modeling
- Custom Visual Design

**Key KPIs**

- Total Sales
- Cost of Goods Sold (COGS)
- Average Growth
- Average Loss
- Gross Sales
- COGS (Previous Year vs Current Year)

All KPIs dynamically update based on applied filters and selections.

**Visualizations Used**

- KPI Cards for Sales, COGS, Avg Growth, and Avg Loss
- Donut Chart showing COGS distribution by country
- Diverging Column Chart for monthly growth vs loss comparison
- Clustered Column Chart comparing COGS (PY vs CY) by month
- Bar Charts with drill-down for COGS and Gross Sales by Segment and Product
- Table with sparklines to visualize product-level sales trends

**Filters & Interactivity**

- The dashboard offers rich interactivity through:
- Product selector
- Segment selector
- Discount band filter
- Month slicer
- Year toggle (2024 / 2025)
- Country selector
- Reset button to clear all selections

All visuals are fully cross-filtered for seamless exploration.

**Light / Dark Mode**

The dashboard includes a Lightbulb toggle button that allows users to seamlessly switch between Light Mode and Dark Mode, enhancing readability and user experience based on preference or environment.

**Design Highlights**

- Clean, modern UI with consistent color logic
- Clear distinction between growth (positive) and loss (negative) values
- Optimized layout for financial storytelling
- User-friendly navigation and interaction

**What I Did**

- Cleaned and prepared raw sales data
- Built a structured and scalable data model
- Developed DAX measures for financial KPIs and time intelligence
- Designed custom visuals for comparative analysis
- Implemented Light/Dark mode using interactive controls

**Project Goal**

To create a business-focused financial dashboard that simplifies complex sales and cost data, enabling faster insights and informed decision-making.
